# Elysium & Abyss

## 🌌 简介
一个融合了地狱与天堂主题的像素风博客，记录我的学习、创作与思考。

## 🧠 功能特性
- 地狱 / 天堂 两种主题切换
- 响应式设计，适配移动端
- 支持 Markdown 编辑和文章发布
- 夜间模式 & 像素 UI

## 📦 使用技术
- 前端：Vue 3, TailwindCSS, Pinia, Vue Router
- 后端：FastAPI (如有)
- 部署：GitHub Pages / Vercel / Netlify（写你的）

```bash
git clone https://github.com/Laugh-Jester/Elysium-Abyss.git
